## 增强方案（按优先级排序）

### 方案一：🔥 粒子拖尾/轨迹效果（最高优先级）
**灵感来源**：VORTEX 项目、Three.js 粒子轨迹+辉光

**效果描述**：
- Hero 区域的火种粒子运动时产生拖尾轨迹
- 拖尾颜色从橙色渐变到透明
- 轨迹长度与粒子速度成正比
- 类似彗星尾巴的效果

**实现方式**：
- 每个粒子维护一个历史位置数组（最近10-20帧）
- 绘制时连接历史位置形成渐变线条
- 使用 `globalCompositeOperation = 'lighter'` 实现发光叠加

**预期效果**：粒子运动时像流星一样拖着橙色光尾，极大增强视觉冲击力

---

### 方案二：🌌 全局背景星座粒子网络（高优先级）
**灵感来源**：粒子星系动画、Flux Visualizer

**效果描述**：
- 在网站所有 section 背后添加一层全局粒子网络
- 70-100 个橙色光点缓慢漂浮
- 距离 < 150px 的粒子之间绘制连线
- 连线透明度随距离衰减
- 鼠标 200px 范围内粒子被吸引
- **新增**：粒子遵循向量场流动（涡旋+螺旋模式）

**实现方式**：
- 创建全局 Canvas 覆盖整个页面（fixed, z-index: 0）
- 使用 Perlin 噪声或三角函数生成向量场
- 粒子受向量场力影响运动
- 保留鼠标吸引效果

**预期效果**：整个网站背景像活的一样，粒子在无形的力量下流动，鼠标经过时产生扰动

---

### 方案三：✨ 鼠标橙色光晕拖尾（高优先级）
**灵感来源**：Cursorly.js、VORTEX 霓虹光标

**效果描述**：
- 鼠标移动时留下橙色光点拖尾
- 拖尾粒子逐渐变小、变透明
- 粒子颜色在橙色系内随机变化
- 移动速度越快，拖尾越长

**实现方式**：
- 创建独立的拖尾 Canvas（z-index: 9998）
- 鼠标移动时生成拖尾粒子
- 粒子有生命周期，逐渐衰减
- 使用 `lighter` 混合模式增强发光感

**预期效果**：鼠标像一支画笔，在页面上画出橙色光迹

---

### 方案四：🎭 增强品牌文案和微交互（中优先级）

**文案增强**：
- Hero 区域第一幕文案优化：
  - "起初，只是几粒漂浮在黑暗中的尘埃" → "起初，只是几粒漂浮在黑暗中的橙色光点"
  - "↓ 向下滚动，点燃它们" → "↓ 向下滚动，让它们汇聚"
  - "后来它们学会了燃烧" → "后来它们汇聚成光"

**微交互增强**：
- 按钮悬停时添加涟漪效果
- 卡片悬停时边框发光脉冲
- 导航链接悬停时下划线从左到右展开（已有，可加强）
- 添加页面切换时的淡入淡出过渡

---

### 方案五：🎯 滚动视差 3D 深度效果（中优先级）
**灵感来源**：Locomotive、Awwwards 获奖网站

**效果描述**：
- 不同层级的元素以不同速度滚动
- 背景装饰元素（deco-circle）移动更慢
- 前景文字移动更快
- 创造深度感

**实现方式**：
- 已有 `data-speed` 属性，可进一步增强
- 添加更多浮动装饰元素
- 使用 CSS `transform: translateZ()` 创建真 3D 深度

---

### 方案六：🌟 成就卡片进入动画增强（中优先级）

**效果描述**：
- 成就卡片进入视口时，从中心向外扩散出现
- 添加轻微的旋转效果
- 数字计数器动画更炫酷（从0翻滚到目标值）

1. **方案一**：粒子拖尾效果（Hero区域）

3. **方案三**：鼠标橙色光晕拖尾
4. **方案四**：文案和微交互增强
5. **方案五**：滚动视差3D深度
6. **方案六**：成就卡片动画增强

## 技术要点

- 所有 Canvas 效果使用 `requestAnimationFrame`
- 使用 `globalCompositeOperation = 'lighter'` 实现发光叠加
- 性能优化：粒子数量根据设备性能动态调整
- 移动端降级：隐藏复杂效果，保持基本功能
- 所有效果支持 `prefers-reduced-motion` 媒体查询
